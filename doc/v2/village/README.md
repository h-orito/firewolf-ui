# 村画面仕様書

## 概要

村画面は、人狼ゲームのメインとなる画面です。プレイヤーはこの画面で村の情報を確認し、他の参加者と発言を通じてコミュニケーションを取りながらゲームを進行します。

## 全体構成

村画面は以下の要素で構成されます：

- **PC画面**: 左側固定メニュー + 右側メインコンテンツ
- **スマホ画面**: ハンバーガーメニュー + メインコンテンツ
- **共通ヘッダーは非表示**（村画面専用レイアウト）

## 詳細仕様

### PC画面レイアウト

#### 左側固定メニュー

固定幅のサイドバーとして表示され、以下の項目を上から順に配置：

1. **[村名](#village-name)**
2. **[村の設定](#village-settings)**
3. **[参加者](#participants)**
4. **[メモ](#memo)**
5. **[ユーザー設定](#user-settings)**
6. **[切り抜き画面リンク](#scrap-link)**
7. **[トップページリンク](#top-link)**
8. **[X(Twitter) ポスト](#twitter-post)**
9. **[ログイン状態](#login-status)**
10. **[広告](#advertisement)**

#### 右側メインコンテンツ

メインコンテンツエリアは以下の構成：

1. **[最上部固定メニュー](#top-fixed-menu)**
2. **[メインコンテンツ](#main-content)**
3. **[最下部固定メニュー](#bottom-fixed-menu)**

### スマホ画面レイアウト

- 左側メニューは通常非表示
- 右側画面最下部固定メニューの左端にハンバーガーメニューボタンを配置
- ハンバーガーメニュー押下時に左側メニューをスライダー表示

## 技術要件

### 使用技術
- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Zustand (状態管理)
- TanStack Query (データフェッチング)

### パフォーマンス要件
- 初期読み込み時間の短縮
- リアルタイム更新（残り時間表示）
- 大量の発言データに対するパフォーマンス最適化

### アクセシビリティ要件
- キーボードナビゲーション対応
- スクリーンリーダー対応
- 適切なARIAラベル設定
