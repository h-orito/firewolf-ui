# FIREWOLF UI v1 ドキュメント

## 概要

FIREWOLF UI v1は、Nuxt.js 2.x + TypeScript + Vue.js 2.xで構築された人狼ゲームのWebアプリケーションです。

## ドキュメント構成

- [技術スタック](./tech-stack.md) - 使用している技術とライブラリ
- [開発ガイド](./development-guide.md) - コンポーネント開発パターンとベストプラクティス
- [API通信](./api-communication.md) - API通信の実装パターン
- [状態管理](./state-management.md) - Vuexストアの使用方法

## プロジェクト情報

- **フレームワーク**: Nuxt.js 2.x（SPAモード）
- **言語**: TypeScript
- **パッケージマネージャー**: npm
- **開発サーバー**: http://localhost:3000
