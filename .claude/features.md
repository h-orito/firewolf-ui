- 村作成
  　- 時間セクション - 開始日時と沈黙時間は「時間」セクションにまとめる - 時間セクションに「1 日の長さは 24 時間固定です」を info として表示 - 開始日時は 7 日後 0 時を初期設定にする - 沈黙時間は 0 を初期設定にする
  - キャラチップセクション
    - API `/charachip/list` でキャラチップ一覧を取得し、キャラチップを選択（複数選択）可能にする
      - 1 つ目を初期選択
    - ダミーキャラをキャラ一覧から選択可能にする
      - API `/charas` で選択したキャラチップ id を指定してキャラ一覧を取得
      - 一人目を初期選択
      - 「画像から選ぶ」ボタンを押すと、モーダルでキャラ画像とともにキャラ一覧を表示し、選択可能
