# FIREWOLF Nuxt4移行 タスク一覧

## 📁 作業ディレクトリ構造

```
firewolf-ui/                 # 作業ディレクトリ (feature/nuxt4ブランチ)
├── .old-nuxt2/             # 退避した旧ファイル群
│   ├── nuxt.config.ts      # 旧Nuxt 2設定
│   ├── package.json        # 旧依存関係
│   └── components/         # 旧コンポーネント（参照用）
├── nuxt.config.ts          # 新Nuxt 4設定
├── package.json            # 新しい依存関係
├── components/             # 新@nuxt/uiコンポーネント
└── ...
```

## 🔄 移行漏れ機能の実装

[x] 秘話相手を選択するプルダウンの横に、画像で選択するボタンを表示し、相手を選択するモーダルを開いて選択可能にする

[x] ユーザー設定で日付を表示するにすると、キャラクター名が長い場合に、キャラクター名が二行になってしまう。キャラクター名は折り返さず、日付の年月と時間のほうが二行になってほしい

[x] 村からトップページに戻って別の村を開くと、サイドバーが開きっぱなしになっている。村を離れたら閉じた状態にしたい。

[x] 発言の内容にNuxt2版と同じsans-serifフォントを適用

[x] 更新ボタン押下後のスクロール開始が早いのか、想定した場所にスクロールされない。発言が読み込まれた後にしたい

[x] 発言抽出モーダルで、発言者や宛先のキャラクター一覧について、スマホ版も2列にしたい

[x] スマホで発言欄にフォーカスしたときに画面が拡大されてしまう。拡大されないようにしたい

[x] プロローグとエピローグでは発言確認の誤爆防止を行わない

[x] 日付リンク押下時など、日付変更時は一番上に遷移させる仕様に統一する（更新ボタン押下は現状通り）

[x] ヘッダーの前日翌日ボタン押下後、一番上にスクロールされない問題を解消
