#### 村作成機能の発言制限設定

**概要**: 村作成画面のルール設定セクションの次に発言制限設定セクションを追加

**詳細タスク**:
1. **MessageRestrictSettingsSection コンポーネント作成**
   - 発言制限設定セクションの実装
   - 説明文の表示（回数・文字数制限の説明）
   - 6種類の発言種別ごとの設定項目表示
   
2. **MessageRestrictSettingItem コンポーネント作成**
   - 発言種別ごとの設定項目コンポーネント
   - 回数・文字数の数値入力フィールド
   - デフォルト値の設定
   
3. **バリデーション機能の実装**
   - Zodスキーマの作成（通常発言は1〜1000、その他は0〜1000、文字数は1〜1000）
   - リアルタイムバリデーション
   - エラーメッセージの表示
   
4. **村作成フォームへの統合**
   - 既存の村作成フォームにセクション追加
   - フォーム状態管理への統合
   - 送信処理への組み込み
   
5. **テスト作成**
   - ユニットテスト（バリデーション）
   - インテグレーションテスト（フォーム統合）

**想定工数**: 1〜2日
**優先度**: 高
**状態**: 完了

## 優先順位

1. **最優先**: bugs.md に記載されたバグ修正
2. **高**: features.md に記載された新機能要望
3. **中**: リファクタリングやパフォーマンス改善
4. **低**: ドキュメント更新

## 注意事項

- 各タスクは適切な粒度でコミットする
- 実装後は必ず lint、format、type-check を実行
- API の型は自動生成されたものを使用
- ドキュメントは実装と並行して更新
- v1 との互換性を意識（URL 構造、API 形式等）
